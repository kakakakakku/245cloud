# Install

```
$ cp config/database.yml.sample config/database.yml
$ $EDITOR config/database.yml
$ bundle install
$ bundle exec rake db:setup
$ cp .env.example .env
  # Edit facebook key
$ $EDITOR .env

$ bundle exec rails s
```

Facebook app idの設定: @pandeiro245 さんにFacebookApp開発用権限の追加依頼する
